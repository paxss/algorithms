### TwoSum
input: int[] **array**, int **target**\
output: [**i**, **j**]\
example: 
```
input: array = [2,7,11,15], target = 9
output: [0,1]
```
**i** and **j** is indexes of the array, array[i] + array[j] must be equal **target**\
[This task on leetcode](https://leetcode.com/problems/two-sum/)
